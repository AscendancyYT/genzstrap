GenZstrap v1.0

Minimal CSS framework. One file. No build step. All vibe.

🚀 Quick Start

<link rel="stylesheet"
  href="https://cdn.jsdelivr.net/gh/ascendancyyt/genzstrap@1.0.0/genzstrap.css">

Then drop in some markup:

<div class="container my-8">
  <h2 class="text-primary fw-bold text-center">Hello GenZstrap</h2>
  <a class="btn btn-primary m-4">Click me</a>
</div>
